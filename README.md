This is an interactive software replicating "Energy-based Haptic Rendering for Real-time Surgical Simulation". 

Required platform: Windows.

Required hardware: Phantom Touch, Nvidia graphical card

Required software: Visual Studio, CUDA  

There is a pre compiled binary file: Bin64/bin/SoftHaptic.exe . 

To execute this file, double click run_script.bat if the Phantom Touch driver and CUDA is installed correctly.
